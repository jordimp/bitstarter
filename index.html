FAQ

What is Release Candidate, Beta, Focus.. what?
Those are different milestones of the game production cycle:
  • Focus Testing: when game is playable, we need input from players. You could shape the game by expressing your opinon or your dislikes about the state of the game at that stage. This helps us a lot to discover early bugs but more important, end user experience by tweaking user interface, game progression, etc.
  • Beta: game is feature complete, but full of bugs. There is a lot to be done yet and focusing on the quality is one of the main focus areas of this milestone. We'd need your help there in order to discover bugs, and change any sloppy part of the game, or some bad designed user interface.
  • Release Candidate: game is ready to be launched.. well, almost... We need some real metrics usage.. actually some huge amount of metrics so we can finally tweak the game and be very sure it's going to be a success!
Why should I back a free to play game?
Because we love to make games that everybody can play.. but we need to pay our bills too.
What are gems?
That would be the currency you could only get by doing and in-app purchase in the game. Very, very, very valuable.
